# Random Password Generator

## This Repository is for the 5th Week challenge

### This application generates a random password in the browser by dynamically updating HTML and CSS by using JavaScript in the background.

#### It generates a password when the `Generate Password` button is clicked.

![image](https://github.com/junushr/random-password-generator/assets/150170720/e04b468b-5b2e-4f3c-bdc3-426436bb97e9)

#### It asks the length of the password we want.

![image](https://github.com/junushr/random-password-generator/assets/150170720/bd0d550c-0b6a-44fc-8212-26d09137e262)

#### If invalid option is given, it will ask again till we select a number between 8 to 128.

![image](https://github.com/junushr/random-password-generator/assets/150170720/bc0cec89-1d72-43c4-a5d5-7ee66857082a)

#### If valid number selected, it will ask if we need choose among the following options:

##### Uppercase Characters
   
![image](https://github.com/junushr/random-password-generator/assets/150170720/c8deee44-c336-4084-8d5a-db32dca44cd5)

##### Lowercase Characters
![image](https://github.com/junushr/random-password-generator/assets/150170720/0107db22-2801-4c8e-949a-6ba226f972df)

##### Numbers
![image](https://github.com/junushr/random-password-generator/assets/150170720/0978eb7d-d750-4ea9-a08a-f2ad53efcc8b)

##### Special Characters
![image](https://github.com/junushr/random-password-generator/assets/150170720/8a3a4055-d1ac-49a5-a828-bd7cdb6d6f34)

##### If we don't select at least one option, it will show a  ask us to select the options again.

![image](https://github.com/junushr/random-password-generator/assets/150170720/c8932e73-c537-4ccf-8945-b93658820831)

#### Based on the options selected by the user it will generate a password and display in the text box and also to the console.

![image](https://github.com/junushr/random-password-generator/assets/150170720/db1c64d1-9625-4e3a-a012-ed815bcf37d8)

![image](https://github.com/junushr/random-password-generator/assets/150170720/aa4f7365-946a-496d-91a7-8980d169fc80)
